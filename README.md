
# Authentication using React and Spring Boot



## Tech Stack

**Frontend:** React, Bootstrap, Node.JS, Axios

**Server:** Spring Boot, Spring Security, JWT, JDBC

## Usage/Examples

**From Terminal**

**Frontend:**

**root directory**:

    cd react-frontend
    npm install
    npm start dev

**Backend:**

**root directory:**

    cd spring-backend
    mvn spring-boot:run
